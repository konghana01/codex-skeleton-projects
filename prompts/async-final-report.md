# 비동기 최종 보고 프롬프트

비동기 작업의 최종 상태를 사용자에게 짧게 보고할 때 사용합니다.

```text
다음 작업의 최종 상태를 사용자에게 짧게 보고해줘.

Task:
- ...

PR:
- ...

Implementation summary:
- ...

Review summary:
- ...

Checks:
- ...

Known risks:
- ...

Output format:
- 5-8줄 이내
- PR URL
- 구현 요약
- 리뷰 반영 여부
- checks
- 남은 risk
- 사용자에게 필요한 next action
```
